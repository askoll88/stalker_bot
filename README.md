# 🌌 S.T.A.L.K.E.R. Chat Bot

Игровой чат-бот для ВКонтакте в стиле S.T.A.L.K.E.R.

## 📍 Локации

- **🏙️ Город** — стартовая локация, хаб
- **🏥 Больница** — лечение персонажа (50 руб.)
- **🛒 Черный рынок** — торговля
- **🔒 Убежище** — личное убежище игрока (открывается позже)

## 📊 Статистика персонажа

- ❤️ Здоровье
- ⚔️ Атака
- 🔋 Усталость
- 💰 Деньги
- ✨ Опыт

## 🚀 Запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Скопируйте `.env.example` в `.env` и заполните данные:
```bash
cp .env.example .env
```

3. Запустите бота:
```bash
python main.py
```

## 🛠️ Технологии

- Python 3
- VK API
- PostgreSQL
